# PdfPage

一个完全在浏览器端运行的 PDF 页面处理工具箱，无需上传服务器，保护你的隐私。

✨ **四大核心功能**：合并 · 拆分 · 旋转 · 删除  
🚀 **纯前端实现**：基于 [pdf-lib](https://unpkg.com/pdf-lib@1.17.1/dist/pdf-lib.min.js) 库，所有操作均在本地完成  
📁 **拖拽上传**：支持点击或拖拽添加文件，体验流畅  
🔄 **灵活旋转**：可按页码范围向左/向右旋转 90°

---

## 📸 功能预览

| 功能 | 说明 | 示例 |
|------|------|------|
| **合并 PDF** | 上传多个 PDF，拖拽调整顺序后合并为一个文件 | 将发票、合同、附件合为一份 |
| **拆分 PDF** | 支持4种模式：按页码范围提取、按固定页数切割、全部拆分为单页、按奇偶页拆分 | 提取第1-3,5,7页或每10页保存一份 |
| **旋转页面** | 指定页面范围（留空表示全部），统一向左或向右旋转90° | 将扫描倒置的页面转正 |
| **删除页面** | 输入要删除的页码，其余页面重新生成PDF | 去掉封面、空白页或特定章节 |

---

## 🧩 技术栈

- [pdf-lib](https://pdf-lib.js.org/) – 创建、修改 PDF 文档
- 原生 HTML/CSS/JavaScript – 无第三方框架，轻量易用
- 完全离线 – 下载后无需联网即可使用

---

## 📦 如何使用


1. 点击本仓库的 `Code` → `Download ZIP` 下载全部文件。
2. 解压后，确保 `离线PDF工具箱.html` 和 `pdf-lib.min.js` 在同一目录。
3. 用浏览器打开 `离线PDF工具箱.html` 即可使用。
