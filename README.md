# 📦 离线PDF工具箱

一个完全运行在浏览器中的 **离线 PDF 全能工具**，无需上传服务器，无需安装软件，保护你的文件隐私。支持 **合并、拆分、旋转、删除页面、PDF 转图片、图片转 PDF、页面可视化重排** 七大功能。

---

## ✨ 为什么选择这个工具？

- 🔒 **纯离线，零上传**：所有操作都在你的浏览器本地完成，文件绝不离开你的电脑。合同、身份证、财务报表，安心处理。
- 🚀 **免安装，开箱即用**：一个 HTML 文件，双击打开就能用。不需要注册、不需要登录、不需要付费。
- 🌐 **全平台兼容**：适用于 Windows、Linux、macOS、安卓、iOS、麒麟等操作系统，只要有浏览器就能运行。
- 💰 **完全免费**：开源且免费，无任何功能限制，无广告，无后门。
- 🎯 **单文件版**：所有依赖库已内联，一个 HTML 文件走天下，无需携带任何 JS 文件。

---

## 🛠️ 功能一览

### 🔗 合并 PDF
- 上传多个 PDF 文件，拖拽调整顺序
- 一键合并下载，30 秒搞定

### ✂️ 拆分 PDF
- **页码范围**：按指定页码提取合并为一个 PDF
- **固定页数**：每 N 页拆分为独立文件
- **全部单页**：每一页都变成独立 PDF
- **奇偶拆分**：奇数页和偶数页分别导出

### 🔄 旋转页面
- 支持向左 90°、向右 90°、旋转 180°
- 可指定页码范围或全部页面
- 专治扫描件"歪脖子"

### 🗑️ 删除页面
- 按页码删除不需要的页面
- 封面、空白页、敏感内容一键移除

### 🖼️ PDF 转图片
- 每页渲染为 JPG 或 PNG 图片
- 可调质量（10%-100%）和分辨率（1×-5×，最高 360 DPI）
- 支持单张下载或批量打包 ZIP

### 🧩 图片转 PDF
- 支持 JPG、PNG、WebP、BMP 等常见格式
- 图片拖拽排序，勾选删除
- 生成 A4 纵向文档，图片自动居中缩放

### 📑 页面重排
- 可视化缩略图网格
- 拖拽调整页面顺序，勾选删除页面
- 可添加空白页，支持多 PDF 混合编排
- 快捷键：`Ctrl+A` 全选，`Delete` 删除选中

## 📖 如何使用

1. 下载 `离线PDF工具箱-单文件版.html`（单文件版已内联所有依赖，无需其他文件）
2. 双击在浏览器中打开
3. 选择对应功能标签，上传文件即可操作

> **推荐浏览器**：Chrome 69+、Firefox 52+、Edge 79+，或任何支持 ES6 的现代浏览器。

## 🏗️ 技术栈

- [PDF.js](https://mozilla.github.io/pdf.js/) - PDF 页面渲染（用于预览和导出图片）
- [PDF-lib](https://pdf-lib.js.org/) - PDF 生成与修改（用于合并、拆分、旋转、删除、重排）
- [JSZip](https://stuk.github.io/jszip/) - ZIP 压缩包生成（用于批量下载图片）
- 原生 Canvas API - 图片处理与缩略图生成

所有依赖已内联至 HTML，无需额外下载。

---

## ⚠️ 注意事项

- 处理超大 PDF 或高分辨率图片时，建议适当降低分辨率倍率，避免浏览器内存溢出
- PDF 转图片功能在预览很多页时会消耗较多性能，建议先用**拆分功能**提取所需页面再转换
- 图片转 PDF 时，超大尺寸图片可能导致生成的 PDF 文件较大，建议提前压缩图片
- 如果遇到任何问题，请在 GitHub Issues 中提交，附上浏览器版本和错误截图
